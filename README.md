- 直接放入游戏文件夹下后打开 MB_Prac.exe

## 功能:
- 按 **backspace** 后
+ **F1** 无敌
+ **F2** 禁用X键
+ **F3** 锁残
+ **F4** 锁P
+ **F5** 永续bgm
- 进入练习界面后可以选择从各种地方开始(甚至居然有ex，，，)
- 更多分辨率
+ 在**config.txt**文件中的第二个值设置为4，则窗口大小是1920x1440(例如 04110)
+ 全屏修复(大概)(测试了10110)
+ 不在窗口上时禁用按键

## 如何播放使用练习器的rep:
- 没法播喵
- 可能之后会加吧大概
